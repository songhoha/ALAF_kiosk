# 🎨 ALAF Client (Frontend)

> **대학 분실물 통합 관리 시스템 - 웹 & 키오스크 인터페이스**

이 디렉토리는 ALAF 프로젝트의 **프론트엔드(React)** 소스 코드를 담고 있습니다.  
사용자 경험(UX) 중심의 웹(Web) 화면과 라즈베리파이 키오스크(Kiosk) 화면을 모두 포함하고 있습니다.

---

## 🚧 개발 진행 상황 (Current Status)

**현재 단계: 프로토타입 & UX 로직 구현 (Prototype Phase)**

현재 버전은 **기능적인 흐름(Flow)과 데이터 로직**을 검증하는 단계입니다.

1.  **UX Flow 중심**: 사용자가 물건을 등록하고 회수하는 논리적 절차 구현 완료
2.  **UI Design**: 기능 테스트를 위한 레이아웃 적용 (최종 디자인 폴리싱 예정)
3.  **Server Integration**: `axios` 설정 완료 (API 엔드포인트 동기화 필요)

## 📂 프로젝트 구조 (Structure)

```bash
src/
├── context/             # ⚡️ 전역 상태 관리 (데이터 공급소)
│   ├── ItemContext.js   # 분실물 데이터 CRUD 및 API 호출 로직
│   └── UserContext.js   # 유저 로그인/인증 로직 (현재 Mock Data)
│
├── ui/                  # 🖼️ 화면 컴포넌트
│   ├── web/             # [PC/Mobile] 일반 웹 브라우저용 페이지
│   │   ├── WebHome.jsx      # 메인: 검색, 카테고리 필터
│   │   ├── WebRegister.jsx  # 등록: 정보 입력 및 이미지 업로드
│   │   └── ...
│   │
│   └── kiosk/           # [Raspberry Pi] 터치스크린 키오스크용 페이지
│       ├── KioskHome.jsx    # 대기 화면
│       ├── KioskCapture.jsx # 웹캠 연동 및 촬영
│       ├── KioskLocker.jsx  # 보관함 제어 시뮬레이션
│       └── ...
│
├── App.js               # 🚦 라우팅 설정 (Web vs Kiosk 경로 분리)
└── App.css              # 🎨 전역 스타일 초기화 (Reset CSS)
```

## 🛠 주요 기능 (Key Features)

### 🖥️ 1. 웹 (Web Interface)
일반 사용자가 PC나 모바일로 접속하는 화면입니다. (`/`)

* **🔍 통합 검색 및 필터링**
    * 물건명, 습득 장소 등으로 키워드 검색 가능
    * 카테고리(전자기기, 지갑, 의류 등)별 모아보기 지원
    * 최신순 / 조회순 정렬 기능
* **📄 상세 정보 확인**
    * 습득된 물건의 사진, 습득 일시, 보관 장소 확인
    * '보관중' 상태와 '수령완료' 상태 구분 표시
* **👤 회원 서비스**
    * **로그인/회원가입**: 기본 인증 절차 UI 및 유효성 검사 로직
    * **마이페이지**: 본인 정보 확인 및 내가 등록한 분실물 관리

### 📟 2. 키오스크 (Kiosk Interface)
라즈베리파이 터치스크린 환경에 최적화된 화면입니다. (`/kiosk`)

* **📸 원스톱 등록 프로세스**
    1.  **정보 입력**: 터치 키패드를 통한 간편 입력
    2.  **사진 촬영**: 웹캠을 연동하여 습득물 실시간 촬영 및 업로드
    3.  **보관함 적재**: 등록 완료 시 특정 보관함이 자동으로 열리는 시뮬레이션
* **🔐 무인 회수 프로세스**
    1.  **본인 인증**: 웹에서 가입한 ID로 로그인
    2.  **물건 선택**: 본인이 수령해야 할 물건 목록 조회
    3.  **수령 완료**: 보관함 타이머 작동 및 수령 완료 처리

## 🚀 실행 방법 (How to Run)

이 프로젝트는 `React`로 개발되었습니다. 터미널에서 `front` 폴더로 이동 후 아래 명령어를 입력하세요.

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 시작
```bash
npm start
```

### 3. 접속 주소 확인
브라우저를 열고 아래 주소로 접속하세요.

| 구분 | URL 주소 | 비고 |
| :--- | :--- | :--- |
| **웹 (Web)** | `http://localhost:3000` | 일반 사용자용 화면 |
| **키오스크 (Kiosk)** | `http://localhost:3000/kiosk` | 라즈베리파이용 화면 |
